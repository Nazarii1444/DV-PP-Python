======================== Інструкція користувача ========================

Задачі лабораторної роботи реалізовані у файлах:
- task1.py
- task2.py
- task3.py
- task4.py
- task5.py

Реалізація стеку у файлі customstack.py
Тести у файлі tests.py.

1. task1.py:
   - Завдання 1.1: Пошук пар дужок у виразі.
   - Завдання 1.2: Сортування пар дужок за позицією відкриття.
   - Завдання 1.3: Сортування пар дужок за позицією закриття.

3. task2.py:
   - Функція calculate_sum_of_nested_list(nested_list) проходить по всіх елементах списку, включаючи вкладені, та обчислює загальну суму.

4. task3.py:
   - Функція validate_expression(expression) перевіряє, чи формула коректно складена, враховуючи дужки та допустимі символи.

5. task4.py:
   - Функція evaluate_expression(expression) виконує обчислення математичних виразів, що містять операції додавання (`S`) та ділення (`D`), використовуючи стек.

6. task5.py:
   - Завдання 5.1: Ітеративне (без рекурсій) вирішення задачі Ханойських веж з використанням стеку.
   - Завдання 5.2: Сортування дисків, що розташовані у випадковому порядку, з використанням трьох стрижнів.

7. tests.py — файл для запуску тестів для всіх завдань:
   - Містить функції для перевірки правильності роботи кожної реалізованої задачі.
   - Функція print_help() показує всі доступні тести та короткий опис кожного.
   - Користувач може запустити певний тест або всі тести одразу.

8. Запуск тестів:
   - Запустіть файл `tests.py`, щоб скористатися зручним інтерфейсом для тестування. У консолі буде відображена допомога з описом кожного тесту.
   - Виберіть номер тесту для запуску або введіть "stop" для виходу з програми.
